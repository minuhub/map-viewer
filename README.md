# Shape File Rendering 프로그램 개발
- 2020.01.06 ~ 2020.02.28
- 지도 플랫폼 개발팀 인턴 프로젝트
- 프로젝트에 관한 자세한 내용은 제 블로그 [삼대이백의 블로그](https://samdae200.tistory.com/19)를 참고하시면 됩니다.

## 사용법
> ./gradlew build (프로젝트 빌드)  
> ./gradlew run (프로젝트 실행)
